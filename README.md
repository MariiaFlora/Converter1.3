# Converter App

Опис проекту: Це проект для конвертації температур між Фаренгейтами та Цельсіями.

## Вимоги

- Java 11 або новіша версія

## Інструкція з запуску

1. Склонуйте репозиторій.
2. Скомпілюйте і запустіть проект за допомогою команд:
   ```bash
   javac app/Main.java
   java app.Main
   ```
